iNtelinet

iConsole

iConsole users can manage Account settings and Add Data , with left side menu it is simple to do. Admin users, those with IADMIN role, additionally administrate Users, Roles and Sources.

Account

Account properties such as password and e-mail can be updated by users.

Change Password

  1. Select Change Password from the iConsole Menu.
  2. Register password information.
    • Current password
    • New password and confirm.
  3. Click <<Save>>.
Note: Password requires at least 8 characters.

Change E-mail

  1. Select Change Email from the iConsole Menu.
  2. Register e-mail information
    • Password
    • New e-mail and confirm
  3. Click <<Save>>.

Users

Set of concessions to consume the Intelinet and iConsole resources. Users may have a role or role but if none of these has sources user won’t be able to log into Intelinet.

Create User

  1. Select Create User from the iConsole.
  2. Register user’s information:
    • User must be unique, only alphanumeric and underscore characters allowed. This is the end user’s login name. Automatically converted to lowercase.
    • First Name mandatory. Automatically converts the first letter to uppercase.
    • Last Name mandatory.
    • Mail
    • Status enabled or disabled, when enabled a mail will be sent with user credentials, if disabled an activation mail.
    • Roles are necessary to feed the Intelinet search, minimum one role required.
      Authorized sources Available sources
      Roles to attach to the new user Roles available
  3. Click <<Save>> to finish registration.
Disabled users can’t log into Intelinet until the activation process is complete.

List Users/Edit Roles

  1. Select List Users from the iConsole Menu.
  2. All users are listed alphabetically, and can be filtered by status (automatically) and/or keyword registered in the search box and click <<Search>>.
  3. Click to edit roles related to the selected user.
  4. Edit Assigned roles and click <<Save>>.
Click to reset search parameters.

Enable Users

  1. Select Enable Users from the iConsole Menu.
  2. Disabled users are listed, check those you want to enable.
  3. Click <<Enable>> button, confirm.

Disable Users

  1. Select Disable User from the iConsole Menu.
  2. Enabled users are listed, check those you want to disable.
  3. Click <<Disable>> button, confirm.
Note: Logged user cannot disable himself and will not appear on the list.

Send activation mail

  1. Select Send activation mail from the iConsole Menu.
  2. Disabled new users are listed, check those you want to activate.
  3. Click <<Send e-mail>> button, confirm.

Delete Users

  1. Select Delete Users from the iConsole Menu.
  2. All users are listed, check those to be deleted.
  3. Click the <<Delete>> button, confirm.
Note: A logged user cannot delete himself and will not appear on the list.

Roles

Set of information sources available for a search.

Create Role

  1. Select Create Role from the iConsole menu.
  2. Register role’s information
    • Role name automatically converted to lowercase.
    • Sources relate one or more sources to the new role,optional.
      Authorized Sources Available Sources
      Sources to relate to the new role. List of all roles available for the new role.
  3. Click <<Save>>.

List Roles

Select List Roles from the iConsole Menu to view all users alphabetically. This grid contains the following features:
  • Click to grant or revoke information sources. Sources in the Authorized Sources list will be available when selected role be assigned.y
  • Click to eliminate role, this operation also removes any former user-role relation, confirm.
    Roles that may leave at least one orphan user, won’t be deleted.
  • Click to list all users that have assigned the selected role. (Read only)
  • Click to list all sources attached to the selected role. (Read only)

Sources

Set of validated information sources that feed a search.

List Sources

Select List Sources from the iConsole Menu to view all roles alphabetically ordered.
  • Click to enable selected source, confirm.
  • Click to disable selected source, confirm.
  • Click to list all related roles of the selected source. (Read only)

Add Data

Users are authorized to upload archives for their enabled sources.
  1. Select Add Data from the iConsole Menu.
  2. Register archives information
    • Select information type .
    • Click and choose archive.
    • Archive description.
  3. Click <<Save>>.
Archives have no size or type restriction.
